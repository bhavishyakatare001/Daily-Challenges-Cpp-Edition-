# Manual Integer Set

**Status:** `Completed`
**Logic:** Linear Search (Loop)

##  Overview
This project is a manual implementation of the **Set** data structure in C++.
It handles uniqueness enforcement manually without using `std::vector` or `std::set`.

##  Code Structure
* **Uniqueness:** Iterates through the array to check for duplicates before inserting.
* **Storage:** Uses a static array `elements[50]`.
